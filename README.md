# Day 1 – Custom String Length Library

## Goal
Implement my own version of C's strlen() without using the standard library.

## Concepts Learned
- Header files
- Include guards
- Multiple source files
- Strings
- Null terminator
- Loops
- GCC compilation
- Git

## Files
include/mystring.h
src/mystring.c
src/main.c

## Build

gcc src/main.c src/mystring.c -Iinclude -o mystring

## Run

./mystring
